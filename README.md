# 🧠 NeuroSymbolic AI: Hybrid Reasoning Engine

## 🚀 Overview
This project implements a **NeuroSymbolic AI system** that combines:
- Machine Learning (Prediction)
- Retrieval-Augmented Generation (RAG)
- Symbolic Rule-Based Reasoning

The system simulates business decisions and provides **explainable AI outputs** using a hybrid reasoning approach.

---

## 💡 Key Features
- 📊 ML-based revenue prediction (Linear Regression)
- 🔎 RAG-based knowledge retrieval using FAISS
- 📐 Rule-based decision engine (Symbolic AI)
- 🧩 Hybrid reasoning combining ML + logic
- 📝 Explainable decision outputs
- 🎥 Interactive dashboard using Streamlit

---

## 🏗️ Architecture

User Input → NLP/Query → RAG → ML Prediction → Rule Engine → Hybrid Reasoning → Decision + Explanation
